# Qwen Configuration Folder

This folder contains Qwen Code configuration, skills, notes, and memory for this project.

## Structure

- `skills/` - Custom skill definitions and templates
- `notes/` - Project notes, learnings, and documentation
- `memory/` - Persistent memory files for context retention
- `config/` - Qwen-specific configuration files

## Usage

- Skills defined here can be referenced in prompts
- Notes help maintain project context across sessions
- Memory files store important observations and patterns
